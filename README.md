# Underwater Object Detection and Perception System

This project implements an underwater image enhancement and object detection pipeline designed for Autonomous Underwater Vehicle (AUV) environments.

## Features
- Noise reduction and color correction
- Contrast enhancement
- Edge detection and threshold-based segmentation
- Object detection optimization for low-visibility environments
